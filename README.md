### 🛒 E-commerce Customer \& Sales Analysis using SQL

##### 📌 Project Overview

This project analyzes an e-commerce database using SQL to understand customer purchasing behavior, product performance, and overall sales
trends. The goal is to derive meaningful business insights that can support data-driven decision-making.

##### Database Structure

The project is built using three relational tables:

* **customers**
* **orders**
* **order\_items**

##### Key Analysis Performed

* Total revenue and sales performance analysis
* Monthly revenue trend analysis
* Top customers based on spending
* Most sold products
* Customer retention (repeat customers)
* Revenue distribution by city and category
* Order status and payment method analysis
* Identification of highest value orders
* Customer ranking using window functions

##### SQL Concepts Used

* JOINs (INNER JOIN)
* GROUP BY \& Aggregations (SUM, COUNT, AVG)
* Subqueries
* Window Functions (RANK)
* Filtering using WHERE \& HAVING

##### Insights

* Electronics category generates the highest revenue
* Repeat customers play a key role in overall revenue
* Certain cities show higher purchasing activity
* Most orders are successfully delivered with limited cancellations

##### Tools Used

* SQL (MySQL)
